This is a sorting visualiser meant for seeing how different sorting algorithms work and what makes some more efficient than others.

Included sorting algorithms:
    - quick sort
    - merge sort
    - radix sort
    - insertion sort
    - selection sort
    - bubble sort

Usage:

It is required that you have python3 installed on your device.

1. Navigate to the directory you have all the downloaded files in

2. Open a terminal in that directory (or open the terminal/command prompt and navigate to the directory through the command line)

3. Run sorter.py with the following command:
    python3 sorter.py
